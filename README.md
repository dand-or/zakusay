# [WIP]zakusay

"zakusay" is an implementation by .NET Core which is a homage to "cowsay".

## Usage

### basic
```
$ zakusay "this is ms06"

\ this is ms06 /
        _____
      /_|_|__\
      |__O__||
     ((([=]))))
```

### commander type option
```
$ zakusay -s "this is commander"

\ this is commander /
       _\____
      /_|\|__\
      |__@__||
     ((([=]))))
```

### another mobilesuit
```
$ zakusay -f dom hello

\ hello /
      ____
    _//  \_`.
   || @) _|| \
   /(\__//))))\
```
